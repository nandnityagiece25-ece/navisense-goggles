# Smart Assistive Navigation Goggles

A wearable assistive technology designed to help visually impaired individuals navigate safely using real-time obstacle detection and intelligent feedback systems.

## 🚀 Overview
This project integrates multiple sensors and embedded processing to create a hands-free navigation system. It detects obstacles, analyzes surroundings, and provides intuitive feedback through vibration and audio alerts.

## 🧠 Key Features
- Real-time obstacle detection using ultrasonic sensors
- Precise distance measurement with IR ToF / LiDAR
- Directional haptic feedback (left, right, front)
- Audio guidance via bone conduction / speaker system
- Lightweight and wearable design
- Works in low-light and complete darkness

## ⚙️ Tech Stack
- ESP32 / Arduino
- Ultrasonic Sensors
- IR ToF / LiDAR
- Embedded Systems
- Haptic Feedback Modules
- Audio Output Systems

## 🔄 Working Principle
1. Sensors continuously scan the environment
2. Microcontroller processes and fuses sensor data
3. Obstacles are classified based on distance and position
4. Feedback is provided:
   - Vibration for directional cues
   - Audio alerts for critical warnings

## 🌍 Applications
- Assistive technology for visually impaired users
- Smart mobility systems
- Wearable navigation devices
